HTML

  • The head has the metadata of the web page.
  • The body contains the actual content of the webpage.
  • head, body, div, ... these are all known as tags.
  • Comments in HTML are done like so ''.

CSS

  • Cascading Style Sheets
  • This styles the content
  • Margin dictates how much space we want around an element.
  • Padding determines the amount of space we want around the content inside an element.

Git

  • Git is used to keep track of software changes during development.
  • Git enables you to develop collaberatively with others.
  • 'git pull' updates your local enviornment using a repo.
  • 'git push' uploads your changes to a repo.
  • Merge Requests are used to review changes before committing them to the main repo

JavaScript

  • Variables allow you to store values in a computer.
  • Control flow; the order in which a computer runs a script.
  • This adds functionality to a website
  • There are premade libraries that you can draw from to add functionality to your code.
